# 硬件调试协议

## 5 阶段调试流程

### Phase 1: 电源检查（必须先做！）
- [ ] 万用表测供电端电压：5V/3.3V 是否在 ±5% 以内
- [ ] 万用表测 LDO 输出是否正常
- [ ] 观察电源指示灯是否亮起
- [ ] 用手背轻触芯片，确认无异常发烫
- [ ] 示波器测电源纹波，< 50mV 为正常

### Phase 2: 地线连续性
- [ ] 万用表蜂鸣档测所有模块 GND 是否连通
- [ ] 确认电源和信号回路没有跨区域
- [ ] 检查模拟地和数字地是否合理分割

### Phase 3: 信号验证
- [ ] 示波器/逻辑分析仪测 I2C SCL/SDA 波形
- [ ] 测 UART TX 是否有数据输出
- [ ] 测 SPI CLK/MISO/MOSI 时序是否匹配
- [ ] 检查中断引脚电平是否正确

### Phase 4: 组件隔离
- [ ] 逐一断开外设模块，确认故障源
- [ ] 最小系统法：只剩 MCU + 电源，看能否工作
- [ ] 替换法：换可疑器件确认

### Phase 5: 软硬分离
- [ ] 烧录 Blink 例程确认 GPIO 正常工作
- [ ] 用串口打印逐步定位故障点
- [ ] 注释掉非关键代码段，逐个功能启用来排查

## 万用表快速指南

| 测量 | 档位 | 预期值 | 说明 |
|------|------|--------|------|
| 供电电压 | DC 20V | 4.75-5.25V 或 3.2-3.45V | LDO输出±5% |
| 短路检查 | 蜂鸣档 | 不叫 | 电源和GND之间不应短路 |
| 二极管 | 二极管档 | 0.4-0.7V | 正向导通反向不通 |
| 电阻 | 200Ω-2kΩ | 标称值±5% | 上拉/分压电阻检查 |
| 电流 | 10A/200mA | 根据设计 | 串入电路测量总功耗 |

## 示波器快速指南

| 信号 | 探头 | 时基 | 幅度 | 预期 |
|------|------|------|------|------|
| I2C SCL | ×1 | 2.5µs/div | 5V/格 | 100kHz/400kHz 方波 |
| UART TX | ×1 | 50µs/div | 5V/格 | 空闲高电平 3.3V/5V |
| PWM | ×1 | 1ms/div | 5V/格 | 占空比可调方波 |
| 电源纹波 | ×10 AC耦合 | 10µs/div | 小信号 | < 50mV |

## 常见 Arduino 编译错误速查

| 错误信息 | 原因 | 修复 |
|----------|------|------|
| `'xxx' was not declared in this scope` | 变量/函数未定义 | 检查拼写和头文件引用 |
| `undefined reference to 'xxx'` | 函数有声明无实现 | 检查库是否正确链接 |
| `expected ';' before 'yyy'` | 漏了分号 | 检查前一行末尾 |
| `no matching function for call to` | 函数参数不匹配 | 检查参数类型和数量 |
| `fatal error: xxx.h: No such file or directory` | 缺少头文件 | 安装对应库 |
| `Board at COM3 is not available` | 端口被占用 | 关其他串口监视器 |
| `A fatal error occurred: Failed to connect to ESP32` | 烧录模式未进入 | 按住BOOT键再按RST |
| `Sketch too big` | Flash空间不足 | 减功能/换大Flash芯片 |
